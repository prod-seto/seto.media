/speckit-plan The application uses Next.js App Router with TypeScript. Data fetching happens in async server components; client components handle state and interactivity only. Data and media files are stored in Supabase (PostgreSQL + Storage). All UI must conform to the Contemporary Soft Club design system: Orbitron, Exo 2, and Share Tech Mono typefaces only, 0px border radius, cool blue-teal palette, ghost-panel and tracklist-panel CSS classes for containers.
